### 客户端命令行操作

启动客户端

```shell
zkCli.sh -server hadoop102:2181
```

#### 命令行语法

| **命令基本语法** | **功能描述**                                                                      |
| ---------------- | --------------------------------------------------------------------------------- |
| help             | 显示所有操作命令                                                                  |
| ls path          | 使用 ls 命令来查看当前znode的子节点 [可监听]  -w 监听子节点变化  -s  附加次级信息 |
| create           | 普通创建  -s 含有序列  -e 临时（重启或者超时消失）                                |
| get path         | 获得节点的值 [可监听]  -w 监听节点内容变化  -s  附加次级信息                      |
| set              | 设置节点的具体值                                                                  |
| stat             | 查看节点状态                                                                      |
| delete           | 删除节点                                                                          |
| deleteall        | 递归删除节点                                                                      |
